### 설치 및 실행

- Front-End

  ```bash
  cd frontend
  npm install
  npm start
  ```

- Back-End

  ```bash
  cd backend
  python -m venv venv
  source venv/Scripts/activate
  pip install -r requirements.txt
  ```
